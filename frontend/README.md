# SkyMind Frontend

Modern flight search interface powered by intelligent AI ranking.

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- SkyMind API

## Development
```bash
npm install
npm run dev
```

Visit `http://localhost:3000`
